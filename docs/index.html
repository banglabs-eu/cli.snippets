01 / Features

Everything for structured note-taking

Capture, cite, tag, search, and export — all from one prompt.

⚡

Instant Capture

Just type text at the prompt — any unrecognized input becomes a note. No commands to memorize for basic note-taking.

📚

Source Citations

Link notes to bibliographic sources with MLA-style citations, authors, publishers, and edition tracking.

🏷

Smart Tagging

Tag notes with comma-separated labels. Autocomplete helps you reuse existing tags. Add or remove tags from any note.

🔖

Locator Tokens

Append p32 or t1:23:45 to notes to automatically parse page and time references into structured metadata.

📄

Markdown Export

Export notes by source, tag, or author as formatted Markdown files. View inline with bat or less.

⇥

Tab Completion

Context-aware autocomplete for commands, sources, tags, authors, and more. The REPL knows what you need.

🔍

Full-text Search

Find notes by content with find or f. Results render as Markdown and open in your pager.

✏

Inline Editing

Edit any note in place with e <id>. Arrow keys to navigate, Enter to save. Tracks last-edited timestamps.

🔒

Multi-user Auth

JWT login with invite-code registration. Per-user data isolation, password changes, and admin invite management.

05 / Getting Started

Three steps to start taking notes

Clone, configure, run.

1

Clone & Install

Clone the repository and install Python dependencies in a virtual environment.

git clone https://github.com/banglabs-eu/cli.snippets.git
cd cli.snippets
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
2

Configure Environment

Copy .env.example to .env and set your backend URL and export directory.

cp .env.example .env

# Edit .env:
BACKEND_URL=http://localhost:8000
EXPORT_DIR=./exports
3

Run

Start the backend, then launch the CLI.

# Backend (separate repo)
docker compose up --build

# CLI
python main.py
